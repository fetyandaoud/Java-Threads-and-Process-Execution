# Java Threads and Process Execution

This project demonstrates basic concepts of process execution and timing in Java.

## Features
- **CommandRunner** – Executes system commands using Java ProcessBuilder.
- **Stopwatch** – Utility class for measuring execution time of processes or commands.
- **TestProcessBuilder** – Example program showing how to run external commands from Java and measure their execution time.

## Technologies
- Java
- ProcessBuilder API
- Basic thread and process handling

## Purpose
This project was created as part of coursework to explore how Java interacts with system processes and how execution time can be measured programmatically.
